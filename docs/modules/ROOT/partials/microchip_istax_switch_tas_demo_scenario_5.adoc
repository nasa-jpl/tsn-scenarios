The following steps enable Time Aware Shaping (TAS) gating on the egress of port 3.  It configures 4 separate exclusively gated traffic classes (1-4), each with a 25 micro-second window, spaced 250 micro-seconds apart, with the remaining time shared by the remaining traffic classes (0, 5-7).
